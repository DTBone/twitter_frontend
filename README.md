## Giới Thiệu

Dự án này là một ứng dụng web clone Twitter với các tính năng chính bao gồm xác thực người dùng, hồ sơ người dùng, đăng tweet, theo dõi/bỏ theo dõi và thích tweet.

## Công Nghệ Sử Dụng

- **Frontend**: ReactJS
- **Backend**: Spring Boot
- **Database**: MySQL
- **Authentication**: JWT (JSON Web Tokens)
- **Styling**: Material UI
